Setting the id on an element helps ajt to recognize it a the same element during diff construction. Element with `id="details-content"` won't be replaced even though its content has changed. Alternatively, `data-ajt-compare` attribute can be utilized instead of `id` This example showcases how `data-ajt-mode="update"` can be utilized to replace the content of an element, that has changed.
